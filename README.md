# 📖 About The Project

**Grade Checker** is a beginner-friendly JavaScript project that allows users to enter their marks and instantly see their grade based on predefined grading rules.

It is built using only **HTML**, **CSS**, and **JavaScript**.

Live link:https://github.com/ridoycoding/Grade-Cheker
